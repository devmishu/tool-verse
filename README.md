# 🚀 ToolVerse – Digital Tools Buying Website

## 📌 Project Description

ToolVerse is a modern web application where users can explore and purchase digital tools. It provides a smooth user experience with product browsing, cart management, and responsive design.

---

## ⚙️ Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- React-Icons
- Lucide-React
- JSON-Data

---

## Key Features

1. Dynamic Cart System (Add, Remove, Checkout)
2. Product & Cart Toggle View
3. Toast Notifications for user actions
4. Displays total cart items count in the navbar cart icon
